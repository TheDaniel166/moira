# Scripts Archive

This folder contains one-shot, historical, or no-longer-maintained scripts from Moira's development history (primarily 2026 native migration, Phase 2/3/4 audits, data substrate work, and debugging sessions).

## Structure

- one_shot_debug_and_diagnostics/ — Ad-hoc debug, diag, check, fetch, and validation scripts that were used once or for specific investigations.
- phase2_migration_benchmarks/ — Specific performance measurement runs from the 2026-05 Phase 2 native migration work (numbers captured in MOIRA_NATIVE_MIGRATION_TRACKER.md).
- phase3_phase4_migration_audits/ — One-time audit and stress/validate scripts from later migration phases.

## Policy
- These are archived for historical reference.
- Do not rely on them for current development.
- Originals were left in the main scripts/ tree in the primary working copy (per collaboration policy at the time of archiving).

See the main docs (MOIRA_NATIVE_MIGRATION_TRACKER.md, etc.) for context on why these existed.
